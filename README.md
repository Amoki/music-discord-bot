Music Discord Bot
================

# Usage

```bash
cp .env.example .env
vi .env # add your token
npm install
npm start
```

# Documentation
The bot reacts to command in public channels.
Supported commands:
- `/join`: the boot join the audio channel you are in.
- `/stop`: stop the music.
- `/next`: play the next music in the queue.
- `/bimdata`: play the Bimdata.io anthem.
