GSCN Discord Bot
================

# Usage

```bash
cp .env.example .env
vi .env # add your token
npm install
npm start
```

# Documentation
The bot reacts to command in public channels.
Supported commands:
- `/je peux aider`: the role `Aidant` will be given to the user
- `/j'ai besoin d'aide`: the role `Besoin d'aide` will be given to the user
